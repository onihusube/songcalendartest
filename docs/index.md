# Anime song CD release date list

個人的なアニメ曲の発売日記録

- ✅ OP/ED
- ✅ キャラソン
- ✅ サントラ
- ✅ BD特典
- ❌ ドラマ・ラジオCD（キャラソン等が含まれていれば載ります）
- ❌ 声優・アーティストのノンタイCD
  
手作業でぽちぽちやっているので更新は遅いです・・・・

## 凡例
